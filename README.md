## Colors

Colors are based off the X11 color names because I wanted it to match my terminal colors with `colorls`. Here's the list of colors that I've used.

- darkcyan `#008b8b`
- lightseagreen  `#20B2AA`
- goldenrod `#DAA520`
- slategray `#708090`
- silver `#c0c0c0`

Non X11 colors
- '#ef6462`
